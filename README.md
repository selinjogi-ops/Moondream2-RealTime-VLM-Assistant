# Moondream2 Real-Time VLM Assistant

A real-time Vision Language Model (VLM) project using Moondream2 that can:

## Features
- Live scene description using webcam
- Text reading from images
- Indian currency note detection
- Multilingual voice output:
  - English
  - Hindi
  - Kannada

## Technologies Used
- Python
- OpenCV
- Moondream2
- PyTorch
- gTTS
- pyttsx3
- Transformers

## Project Purpose
This project was developed while mentoring students at Indus International School Bangalore (IISB) to demonstrate how Vision Language Models work in real-world applications.
